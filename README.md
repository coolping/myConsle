# Serial Loop Monitor Tool

## Overview

這是一個給 DQA / Bring-up / FW Debug 使用的 Python 串口監控工具。

功能：

- Serial Monitor
- HEX 顯示
- ANSI 彩色支援
- 自動 reset loop
- 關鍵字觸發
- CLI 互動控制
- 自動循環計數
- ANSI escape ignore keyword match

---

# Install

```bash
pip install pyserial colorama
```

---

# Run

```bash
python serial_loop_monitor.py
```
